# Monsters-of-the-Devastation
 Game created with p5play\
Play game [here](https://editor.p5js.org/CheesyModz/sketches/qxQ6g5jR9)

# Controls
`w` to jump\
`a` to move left\
`d` to move right\
`j` to punch\
`k` to double punch\
`l` to throw rocks\
